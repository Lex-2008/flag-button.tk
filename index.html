This site provides a simple geolocation API for Flag Button extension for Opera browser. It takes information from several sources and returns the best result. Sources are:

  • freegeoip.net — uses free GeoLite database by MaxMind.
  • ipgeobase.ru — provides geolocation information for Russian and Ukrainian IPs.
  • ipinfodb.com — uses free IP2Location Lite database by IP2Location.
  • quova.com — Neustar IP Intelligence
  • maxmind.com — Local copy of GeoLite City data by MaxMind, downloaded at 2013-01-12
